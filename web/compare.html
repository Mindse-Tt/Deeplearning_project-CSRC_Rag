📘 怎么读这个对比?

  • G0(左) 是基线: 裸 Qwen-0.5B,没有证据,prompt 很弱。典型失效模式 = 拒答 / 自由发挥 / 编造人名公司名。
  • G3(右) 是本项目产出: 同一基座 + LoRA 微调 + 相同检索证据 + 强约束 prompt。学会了引用 [EventID=xxx] + 拒绝证据外的推断。
  • G1/G2 是消融对照: 说明光加 RAG(G1)或光加强 prompt(G2)都无法让小模型 zero-shot 遵循格式指令 — 必须要 LoRA 微调才能达到 G3 的结构化输出。
  • 红底黄字高亮的是证据里不存在但答案里出现的数字(幻觉); 绿色 [EventID=...] 是格式合规的引用。
