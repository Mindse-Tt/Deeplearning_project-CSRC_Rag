# 前端测试问题

以下问题可直接用于后续前端联调和答辩前测试。

## 1. 案例检索类

- 这段违规行为最像哪些历史处罚案例？
- 上市公司董事利用内幕信息买入股票获利，历史上通常对应哪些案例？
- 信息披露违规并涉及误导性陈述的案件，近几年有哪些典型案例？

## 2. 法规依据类

- 根据这段违规行为，通常违反哪些法律法规？
- 相似案例里主要援引了哪些法条？
- 这类案件常见的法规依据是什么？

## 3. 处罚推荐类

- 根据这段违规行为推荐处罚方式，并说明法条依据。
- 如果是上市公司高管参与内幕交易，通常会出现哪些处罚方式？
- 这种行为是否可能触发市场禁入？

## 4. 趋势分析类

- 近五年内幕交易相关案件的处罚是否更严？
- 不同监管机构对信息披露违规的处罚方式有什么差异？
- 上市公司相关案件最常见的处罚组合是什么？

## 5. 用于首轮演示的示例问题

建议你在第一版前端中优先放下面这三条：

1. 根据这段违规行为推荐处罚方式，并说明法条依据。
2. 上市公司董事利用内幕信息买入股票获利，历史上通常对应哪些案例？
3. 信息披露违规案件常见的法规依据和处罚方式是什么？

这三条分别覆盖：

- 处罚推荐
- 相似案例检索
- 法规依据问答
